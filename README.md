# Bin Extractor

An Android app to extract/decompress `.bin` files.

## Features

- Browse and select `.bin` files on device
- Extract/decompress content
- Save extracted files to a chosen output directory
- Minimal UI, lightweight

## Build

```bash
./gradlew assembleRelease
```

## Requirements

- Android 6.0+ (API 23)
- Storage read/write permissions
